## 安装
```bash
git clone https://github.com/meowrain/gpt-translator-cli
cd gpt-translator-cli
npm install
```
## 配置

1. 在 OpenAI 网站上申请 API KEY 和 API URL。
2. 配置apikey和apiurl

执行 `npm run config` 进行配置


## 运行

```bash
npm run fy
```

运行时，机器人将会提示你输入问题并回答。

## 项目结构

index.js: 程序入口文件
config/: 配置文件目录
utils/: 工具函数目录
